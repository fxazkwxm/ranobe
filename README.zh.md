### 功能特色
- 阅读轻小说
- 无广告、无付费墙
- 无限小说浏览
- 多语言来源
  - 英文
  - 俄文
- 支持关键词搜索
- 章节搜索与排序
- 连续纵向阅读
- 支持浅色和深色模式
- 个性化阅读体验
- 自定义字体大小
- 自定义背景和字体颜色
